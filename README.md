# Marknotes Api
> A api for app marknotes (An markdown notes)

## Environment
- Node 21.x
- Postgres 16.x

## How to run
1. Install dependencies
```bash
npm install
```

2. Run docker db
```bash
docker compose up -d
```

3. Run app
```bash
npm run start:dev
```

## Stacks
- [BiomeJs](https://biomejs.dev)
- [TypeORM](https://typeorm.io)
- [RestClient](https://github.com/Huachao/vscode-restclient)
- NestJs
- Docker
